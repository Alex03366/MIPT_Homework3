# Books Scraper

Парсер для сбора данных о книгах с сайта Books to Scrape.

## Цель проекта
Сбор и анализ данных о книгах.

## Функциональность
- Парсинг данных о книгах
- Сбор данных со всех страниц каталога  
- Автоматический ежедневный сбор данных
- Сохранение результатов в файл
- Автотесты

## Используемые библиотеки
- requests
- beautifulsoup4  
- schedule
- pytest

## Инструкция по запуску
1. `pip install -r requirements.txt`
2. `python scraper.py`
3. `pytest tests/ -v`
